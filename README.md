Flutter + Lcov for code Coverage reports!

## Usage

### command line

*build for android*

```bash
docker run --rm \
  -v "$(pwd):/src/" \
  gableroux/flutter:0.0.13 \
  bash -c "cd /src/ && flutter build apk --release"
```

*run the tests*

```bash
docker run --rm \
  -v "$(pwd):/src/" \
  gableroux/flutter:0.0.13 \
  bash -c "cd /src/ && flutter test --coverage && genhtml coverage/lcov.info --output=coverage"
```
